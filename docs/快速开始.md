# 🚀 V2简化版系统 - 快速开始

## 一、双系统架构

本目录包含**两个互补的监控系统**：

| 系统 | 文件 | 适用场景 | 速度 |
|------|------|----------|------|
| **UIA监控** | `monitor.py` | 大部分Windows软件 | 0.5秒 |
| **视觉识别** | `vision_monitor.py` | 微信、Blender等自绘UI软件 | 1-2秒 |

**⚠️ 优先级原则**：
1. **默认只用UIA**：`latest.json` + `latest.png`（覆盖90%场景）
2. **仅当UIA失败时**才切换到Vision：`latest_vision.json`
3. 失败条件：`control_count < 5` 或 找不到目标元素

## 三、启动监控

### 方式1：UIA监控（默认）
```bash
cd f:\github\ai浏览器自动化\v2_simplified
python monitor.py
```

### 方式2：视觉识别（UIA失效时）
```bash
python vision_monitor.py
# 或启用Florence-2获得更好的图标识别
python vision_monitor.py --florence
```

### 自定义间隔
```bash
python monitor.py --interval 0.3        # UIA 0.3秒更新
python vision_monitor.py --interval 1.5  # 视觉 1.5秒更新
```

## 五、AI操作流程

### 标准操作循环
```
1. 确认监控系统运行中
2. 读取 latest.json 获取目标控件坐标
3. 执行 pyautogui.click(x, y)
4. 查看 latest.png 截图验证效果 ← 每次操作后必须执行
5. 根据截图决定下一步操作
```

### 命令示例
```bash
# 点击
python -c "import pyautogui; pyautogui.click(500, 300)"

# 中文输入
python -c "import pyperclip; pyperclip.copy('中文'); import pyautogui; pyautogui.hotkey('ctrl', 'v')"

# 按键
python -c "import pyautogui; pyautogui.press('enter')"

# 快捷键
python -c "import pyautogui; pyautogui.hotkey('ctrl', 's')"
```

## 七、常见问题

### Q: JSON没有更新？
A: 检查监控系统是否在运行，检查当前窗口是否是目标窗口

### Q: 找不到目标控件？
A: 控件可能不可见或名称不同，尝试用截图查看实际界面

### Q: 点击没反应？
A: 重新获取坐标，确认窗口在前台，重试操作
